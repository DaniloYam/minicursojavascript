# Minicurso de JavaScript Básico

Bem-vindo ao **Minicurso de JavaScript Básico**!  
Este curso é destinado a iniciantes e ensina conceitos fundamentais de JavaScript por meio de PDFs de cada módulo. Todo o conteúdo é online, gratuito e focado em **arquivos de texto e PDFs**, sem necessidade de vídeos ou gravações.

---

## **Índice dos Módulos**

1. [Módulo 1 – Introdução](pdfs/modulo1-introducao.pdf)  
   Apresentação do JavaScript, execução de código, primeiras linhas e ambiente de desenvolvimento.

2. [Módulo 2 – Variáveis e Tipos de Dados](pdfs/modulo2-variaveis.pdf)  
   Números, strings, booleanos, constantes e operadores.

3. [Módulo 3 – Estruturas Condicionais](pdfs/modulo3-condicionais.pdf)  
   Uso de `if`, `else`, `switch` e operadores lógicos.

4. [Módulo 4 – Laços de Repetição](pdfs/modulo4-lacos.pdf)  
   Loops `for`, `while`, `do...while` e `for...of`.

5. [Módulo 5 – Funções](pdfs/modulo5-funcoes.pdf)  
   Criação de funções, parâmetros, retorno, funções anônimas e arrow functions.

6. [Módulo 6 – Arrays e Objetos](pdfs/modulo6-arrays-objetos.pdf)  
   Listas de dados, objetos, arrays de objetos e manipulação básica.

7. [Módulo 7 – Funções Avançadas e Manipulação de Arrays](pdfs/modulo7-avancado.pdf)  
   Métodos como `map`, `filter`, `reduce`, `find`, `some`, `every` e boas práticas.

---

## **Como usar o minicurso**

1. Baixe os PDFs da pasta [pdfs](pdfs) do repositório.  
2. Abra os PDFs no seu leitor de arquivos preferido.  
3. Siga a ordem dos módulos para aprender de forma progressiva.  
4. Pratique escrevendo código no seu computador usando qualquer editor (VSCode, Sublime Text, etc).
